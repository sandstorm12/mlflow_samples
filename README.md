# Sample_mlflow

A sample project to act as a learning tool and a baseline for future mlflow-based AIML projects


## Usage

Launch MLFlow UI:

```bash
# Launch using default parameters
mlflow ui

# Launch on custom address and port
mlflow ui -h 0.0.0.0 -p 5000
```
